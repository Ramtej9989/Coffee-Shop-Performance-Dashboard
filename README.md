## Overview
This Power BI dashboard analyzes coffee shop sales performance, customer behavior, and profitability across regions and time periods. It helps stakeholders track key business metrics and identify growth opportunities.

## Key KPIs
- Total Revenue
- Total Profit
- Profit Margin %
- Total Transactions
- Average Order Value
- Weekend vs Weekday Revenue %

## Dashboard Features
- Monthly revenue trend analysis
- Revenue and profit by product category
- Top 10 products by revenue
- Regional performance comparison
- Weekend vs weekday sales analysis
- Customer segmentation by age group, gender, and loyalty tier
- Store-level performance summary

## Tools & Technologies
- Power BI
- DAX
- SQL
- Excel

## Data
- Public / mock dataset
- Cleaned and modeled for analytical purposes
- Optimized data model for performance

## Key Insights
- Weekday revenue is significantly higher than weekend revenue
- Coffee products generate the highest revenue
- Certain regions and stores consistently outperform others
- Loyalty tiers contribute evenly to overall revenue

Due to Power BI Service publishing limitations, the dashboard is shared through screenshots and documentation. The Power BI (.pbix) file and sample dataset are included for review.
